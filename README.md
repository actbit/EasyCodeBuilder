# EasyCodeBuilder
## 機能
 GUIでC#のコードを生成できます。
 コードの保存、コードをコンパイル、実行することが可能です。
 条件分岐、Console.WriteLineメソッド、Console.ReadLine、キャスト
 代入、配列、変数に対応しています。
## 操作
 exeを実行するとWindowが開き左側にコントロールを追加していき
 **コードを生成** を押すことでコードを生成できます。
 コード生成をするとコードを生成できます。
 コードを生成後**実行ファイルを保存** を押すことでexeを生成できます
 コードを生成後**実行する** を押すことで実行できます
